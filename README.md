# MOIL Http Server

A Node.js web app used to generate MOIL anypoint matrices. 

## 1.   Download 

   Pull the repo

   - *git clone https://github.com/yourskc/moil_httpserver.git*

   - *cd moil_httpserver*

## 2.   Run

   - *npm install*

   - *node index.js*
